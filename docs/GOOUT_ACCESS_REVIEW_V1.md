# GoOut Access Review V1

## Overený stav

- verejná dokumentácia API nebola potvrdená
- žiadosť o partnerský alebo feed prístup zatiaľ nebola odoslaná
- priame kopírovanie a opätovné zverejňovanie obsahu nebude použité
- konektor `goout_partner` zostáva vypnutý
- automatický Cron pre GoOut nie je povolený

## Povolený budúci postup

GoOut konektor sa môže zapnúť iba po získaní:

1. písomného súhlasu,
2. schváleného API alebo feedu,
3. pravidiel atribúcie,
4. limitov používania,
5. povolenia zobrazovať obrázky, popisy, ceny a odkazy.

Technická vrstva `partner-ticket-sync` je pripravená na autorizovaný feed.
