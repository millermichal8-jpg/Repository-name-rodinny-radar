# Návrh žiadosti pre Predpredaj.sk

**Stav:** pripravený návrh, neodoslané

**Komu:** obchod@predpredaj.sk

**Predmet:** Partnerské dátové prepojenie – Rodinný radar

Dobrý deň,

pripravujeme mobilnú aplikáciu Rodinný radar, ktorá rodičom pomáha vyhľadávať
aktuálne rodinné, kultúrne a voľnočasové podujatia podľa lokality, termínu
a veku detí.

Radi by sme sa informovali o možnosti autorizovaného dátového prepojenia
s Predpredaj.sk.

Potrebovali by sme vedieť:

- či poskytujete partnerské API alebo dátový feed,
- ktoré informácie môžeme zobrazovať,
- aké sú pravidlá atribúcie a používania obrázkov,
- aké sú limity synchronizácie,
- či je možné používať partnerské odkazy na nákup vstupeniek,
- aké obchodné alebo technické podmienky musíme splniť.

Bez vášho súhlasu neplánujeme automaticky získavať ani opätovne publikovať
obsah portálu.

Ďakujeme za informácie.
