# Predpredaj.sk Access Review V1

## Overený stav

- Predpredaj.sk ponúka obchodnú spoluprácu organizátorom.
- Verejná dokumentácia API alebo dátového feedu nebola potvrdená.
- Žiadosť o dátový prístup zatiaľ nebola odoslaná.
- Automatické získavanie a opätovné publikovanie obsahu nebude zapnuté bez písomného súhlasu.
- Produkčný konektor musí zostať vypnutý, kým nie sú známe technické a licenčné pravidlá.

## Potrebujeme potvrdiť

1. dostupnosť API, XML, JSON, CSV alebo iného feedu,
2. povolené použitie názvu, dátumu, miesta, cien, popisu a obrázkov,
3. povinnú atribúciu a spätný odkaz,
4. frekvenciu synchronizácie a limity,
5. informácie o zrušených alebo presunutých podujatiach,
6. možnosť partnerských alebo affiliate odkazov.

## Bezpečný stav

Do získania písomného súhlasu:

- konektor zostane vypnutý,
- nebude zaradený do Cronu,
- nebude vykonávať scraping,
- nevykoná žiadny zápis do produkčného katalógu.
